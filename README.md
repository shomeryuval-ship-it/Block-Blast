# Block-Blast
Block Blast game
מטרת המשחק: 
להשיג את הניקוד הגבוה ביותר על ידי השלמת שורות ועמודות מלאות על לוח.
חוקי המשחק:
בכל תור תקבל 3 צורות בתחתית המסך.
גרור את הצורה באמצעות האצבע ומקם אותה במשבצות הפנויות שעל הלוח.
כאשר שורה אופקית או עמודה אנכית מתמלאת לחלוטין - היא מתפוצצת, נעלמת ומעניקה לך נקודות בונוס.
לאחר שתשתמש בכל 3 הצורות, תקבל 3 צורות חדשות. 
המשחק מסתיים כאשר אין יותר מקום פנוי על הלוח לאף אחת מהצורות שנותרו לך במאגר. 
תכנן את הצעדים שלך בחוכמה.
